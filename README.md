# gif_compressor.py
Herramienta CLI en Python para compresión masiva de GIFs. Utiliza ingeniería inversa (Explode-Merge) con Gifsicle para reducir redundancia temporal (50% frames) y optimizar la paleta globalmente. Ideal para firmas de email y vectores, garantizando texto nítido &lt;500KB. Incluye auto-diagnóstico y UI con barras de progreso.
